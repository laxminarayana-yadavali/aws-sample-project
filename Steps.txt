Step 1: create a IAM Role
a. Open IAM 
b. click on roles
c. create a new role
d. add policies i. dynamodbfullaccess ii. lambdainvokefunction iii. lambdaBasicExecutionRole
e.  give a name and create role

Step 2: Create a lambda function
a. Open lambda
b. Create Function
c. Runtime-> python 3.8
d. use existing role( IAM role created previously)
e. Zip all the three files(rename with fun name) and upload at function and deploy the function code

Step 3: Create a DynamoDB Table
a. give a table name and partition key and use default settings
b. click on create table 

Step 4: Create an API
a. Go to API Gateway
b. create an API with Rest API
c. create a GET method and select the lambda function created earlier and enable lambda proxy integration
d. create a POST method and select the lambda function created earlier and enable lambda proxy integration

Step 5: Deploy API
a. deploy api by creating a stage(stage name= dev) 
b. copy the invoke url and paste it on a browser and test it by giving the inputs.
